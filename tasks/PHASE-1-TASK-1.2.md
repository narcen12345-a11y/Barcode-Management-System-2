# PHASE 1 - TASK 1.2
## Enterprise Architecture

### Tujuan

Mempersiapkan struktur arsitektur project agar siap dikembangkan menjadi aplikasi enterprise sesuai PRD.

### Instruksi

Baca seluruh folder `.ai`.

Baca seluruh folder `docs`.

Kerjakan hanya task ini.

### Yang harus dilakukan

Tambahkan folder berikut di dalam `backend/app` apabila belum tersedia.

Actions

DTOs

Enums

Exceptions

Helpers

Interfaces

Policies

Repositories

Rules

Services

Traits

ValueObjects

Support

Observers

Events

Listeners

Jobs

Notifications

Mail

Setiap folder harus memiliki file `.gitkeep` agar tetap tersimpan pada Git.

Jangan membuat business logic.

Jangan membuat Controller.

Jangan membuat API.

Jangan membuat Authentication.

Jangan membuat Database baru.

Jangan mengubah migration.

Jangan menghapus file yang sudah ada.

### Setelah selesai

Jelaskan:

- Folder apa saja yang dibuat.
- Mengapa folder tersebut diperlukan.
- Apakah ada konflik dengan struktur Laravel.

Kemudian berhenti dan menunggu Task berikutnya.

## Definition of Done

- Semua folder berhasil dibuat.
- Tidak ada kode bisnis yang ditambahkan.
- Struktur Laravel tetap dapat dijalankan.
- Tidak ada error baru.
