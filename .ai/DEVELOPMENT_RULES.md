# DEVELOPMENT RULES

Gunakan:

Backend
Laravel 12

Frontend
React + Vite

Database
PostgreSQL

Styling
TailwindCSS

UI
Shadcn UI

Barcode
Code128

Authentication
Laravel Sanctum

Pattern

Repository Pattern

Service Layer

REST API

RBAC

Semua perubahan harus:

- Tidak merusak fitur lama
- Memiliki validasi
- Menggunakan migration
- Menggunakan seeder jika diperlukan
- Menggunakan clean code
- Mengikuti SOLID Principle

Jangan membuat shortcut.

Selalu buat struktur yang mudah dikembangkan.
