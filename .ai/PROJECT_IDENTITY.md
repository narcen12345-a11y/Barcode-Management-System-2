# PROJECT IDENTITY

Nama Project:
Barcode Management System

Tujuan:

Membangun aplikasi berbasis web yang digunakan untuk membuat Barcode Code128, menyimpan seluruh informasi material, mencatat seluruh aktivitas secara otomatis, menyediakan histori perubahan data, pencarian cepat, filter data, serta manajemen user dengan sistem approval.

Project ini menggunakan AI sebagai programmer utama.

Seluruh pengembangan HARUS mengikuti PRD.

Semua keputusan coding harus mengutamakan:

- Kemudahan Maintenance
- Keamanan
- Scalability
- Clean Code
- Modular

Target User:

- Admin
- Warehouse
- Field Engineer
- Asset Management

Project ini diperkirakan akan terus berkembang sehingga struktur project HARUS scalable.
